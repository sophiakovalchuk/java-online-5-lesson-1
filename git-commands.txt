git init .  - Ініціалізація репозиторія

git status  - щоб зрозуміти поточний стан репозиторія

git add <імʼя файлу> - Щоб додати файл в staging area

git commit -m "<опис коміту>" - Щоб зробити commit

git checkout <хеш коміта> - відкочуємось назад, щоб перейти до конкретного коміту

clear - очищає сторінку терміналу

git log - щоб подивитися історію комітів

git checkout master - переходимо на гілку master
